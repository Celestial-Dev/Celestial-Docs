# Bookmarklets
Bookmarklets are a bookmarks that contains JavaScript commands that add new features to the browser. When clicked, a bookmarklet
performs one of a wide variety of operations, such as running a search query or extracting data from a table. For example, clicking on a bookmarklet after selecting
text on a webpage could run an Internet search on the selected text and display a search engine results page.

HOW TO ADD: Copy and paste the bookmarklet boxes into your bookmark bar (located under your URL box), and they will be added. You can right-click them and press "Edit"
to change the bookmark name and URL.
