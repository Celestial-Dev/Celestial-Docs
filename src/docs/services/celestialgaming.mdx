# Celestial Gaming
Celestial Gaming is a free to use, open-source, gaming site with a large collection of html5, flash, mobile, and retro games to play
out of the box in your browser! WARNING: I will not take responsibility if this becomes a distraction in the classroom, nor will I take
it if this gets a student in trouble.

## Setup Guide

### Prerequisites: Must have [NodeJS](https://nodejs.org/en/) installed!!

### Step 1: Downloading the Files

```
$ git clone https://github.com/Celestial-Dev/Celestial-Gaming
$ cd Celestial-Gaming
```

### Step 2: Installing Packages

```
$ npm install
```

### Step 3: Starting It
If you want to run a developer build, do:
```
$ npm run dev
```
If you want to build & run this, do:
```
$ npm run build
$ npm run start
```
